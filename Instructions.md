# Main.java

The main file contains the culmination of the three folders of our program. It imports packages from the StudentInformation, CollegeInformation, and Math folders.

This means that if any errors arise, you can tell which section the error is coming from based off of which package is throwing the error.

To use the main program, run the following commands in the project folder's directory.

javac -cp ".:gson-2.10.1.jar" */*.java Main.java
java -cp ".:gson-2.10.1.jar" Main

To see if you are in the right folder, the terminal should read
GrandChampionsCSIndStud2526 % 
Right at the end.

If it does not match that then you are in the wrong folder and should search up a tutorial on terminal navigation.

# Tutorial

The tutorial folder contains a basic scanner for the tutorial file. To run this tutorial, open a terminal and route it to the Tutorial folder. If you don't know how to navigate a terminal, search it up or chatgpt it.

Once you are in the tutorial folder, run the following commands:

javac Tutorial.java
java Tutorial.java

Afterwards, the prompt should come up, you input your age, and the output should follow.

To see if you are in the right folder, the terminal should read
Tutorial % 
Right at the end.

If not, you are in the wrong folder. Search up a tutorial on how to route into the folder or ask ChatGPT.
