# Tutorial

The tutorial folder contains a basic scanner for the tutorial file. To run this tutorial, open a terminal and route it to the Tutorial folder. If you don't know how to navigate a terminal, search it up or chatgpt it.

Once you are in the tutorial folder, run the following commands:

javac Tutorial.java
java Tutorial.java

Afterwards, the prompt should come up, you input your age, and the output should follow.

to check if your terminal is in the right folder, run the following command:
ls

It should output
Main.class      Main.java       README.md

If not, you are in the wrong folder. Search up a tutorial on how to route into the folder or ask ChatGPT.
